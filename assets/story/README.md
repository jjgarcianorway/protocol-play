# Story Images

Image placeholders for protocol: play story scenes.
Each `.prompt.txt` file contains a detailed prompt for AI image generation.

Generate images using ChatGPT/DALL-E, Nano Banana 2, Midjourney, or similar.
Save as PNG, recommended size 1920x1080 or 1280x720.
The game will load them during dialog scenes.

## Art Style Guide
- Low-poly 3D feel OR painterly concept art
- Soft, cinematic lighting
- Color palette: teals, blues, warm ambers
- NOT photorealistic — stylized, evocative
- Mood: melancholic beauty, quiet hope
