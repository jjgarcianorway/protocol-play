# protocol: play

A game built with [Bevy](https://bevyengine.org/) (Rust).

## Status

**v0.1.0** - Initial project setup. Minimal Bevy application with a window. The foundation is in place, ready for gameplay development.

## Building

Requires [Rust](https://www.rust-lang.org/tools/install).

```sh
cargo build --release
```

The binary will be at `target/release/protocol-play`.

## Running

```sh
cargo run --release
```
